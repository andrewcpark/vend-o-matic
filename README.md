# vend-o-matic
A vending machine web application
[Check demo here](https://vendingmachine-vendomatic.herokuapp.com/)

## Getting Started

### Clone this repository
```bash
git clone https://github.com/andrewcpark/vend-o-matic.git
```

### Install dependencies
```bash
npm install
```

### Run Server Only
```bash
npm run server
```
